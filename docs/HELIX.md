# Helix Cheatsheet

Modal editor — three main modes: **Normal** (default), **Insert**, **Select**.

---

## Modes

| Key | Action |
|---|---|
| `i` | Enter Insert mode (before cursor) |
| `a` | Enter Insert mode (after cursor) |
| `o` / `O` | New line below / above and enter Insert |
| `v` | Enter Select (character) mode |
| `V` | Select current line |
| `Esc` | Return to Normal mode |

---

## Navigation

| Key | Action |
|---|---|
| `h j k l` | Left / down / up / right |
| `w` / `b` | Next / previous word |
| `gg` / `ge` | Start / end of file |
| `gh` / `gl` | Start / end of line |
| `Ctrl+d` / `Ctrl+u` | Scroll half page down / up |
| `{` / `}` | Previous / next blank line |

---

## Editing

| Key | Action |
|---|---|
| `d` | Delete selection |
| `c` | Change selection (delete + Insert mode) |
| `y` | Yank (copy) |
| `p` / `P` | Paste after / before |
| `u` / `U` | Undo / redo |
| `>` / `<` | Indent / dedent selection |
| `~` | Toggle case |
| `Ctrl+c` | Comment / uncomment line |

---

## Search & Navigation

| Key | Action |
|---|---|
| `/` | Search forward |
| `?` | Search backward |
| `n` / `N` | Next / previous match |
| `*` | Search for word under cursor |
| `f<char>` | Jump to next occurrence of char on line |
| `gd` | Go to definition (LSP) |
| `gr` | Go to references (LSP) |
| `gy` | Go to type definition (LSP) |

---

## File & Buffer Management (Space menu)

| Key | Action |
|---|---|
| `Space+f` | File picker |
| `Space+b` | Buffer picker |
| `Space+/` | Global search (ripgrep) |
| `Space+d` | View LSP diagnostics |
| `Space+a` | LSP code actions |
| `Space+r` | Rename symbol (LSP) |
| `Space+k` | Hover docs (LSP) — also just `K` |

---

## LSP

| Key | Action |
|---|---|
| `K` | Hover documentation |
| `gd` | Go to definition |
| `[d` / `]d` | Previous / next diagnostic |
| `Space+d` | Diagnostics list |
| `Space+a` | Code actions |
| `Space+r` | Rename symbol |
| `Ctrl+Space` | Trigger autocomplete |

---

## Windows & Splits

| Key | Action |
|---|---|
| `Ctrl+w+v` | Split vertical |
| `Ctrl+w+s` | Split horizontal |
| `Ctrl+w+h/j/k/l` | Move between splits |
| `Ctrl+w+q` | Close split |

---

## Commands (`:`)

| Command | Action |
|---|---|
| `:w` | Save (auto-formats via ruff) |
| `:q` | Quit |
| `:wq` | Save and quit |
| `:q!` | Quit without saving |
| `:theme <name>` | Switch theme (try `:theme tokyonight`) |
| `:set line-number absolute` | Toggle line number style |
| `:lang` | Show current language |

---

## Config

| Location | Purpose |
|---|---|
| `~/repos/nixos-config/home/modules/helix.nix` | Settings, theme, language servers |
| `~/.config/helix/config.toml` | Generated by home-manager (read-only) |
| `~/.config/helix/languages.toml` | Generated by home-manager (read-only) |

Edit `helix.nix` and run `nixos-switch` to apply changes.
